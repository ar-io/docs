## Adding ARIO to Your Wallet

ARIO is a standard SPL Token on Solana. Most Solana wallets auto-detect ARIO once you receive tokens, but you can also add it manually using the mint address.

## Supported Wallets

### Phantom

[Phantom](https://phantom.app) is the most popular Solana wallet, available as a browser extension and mobile app.

<Steps>
  <Step>
    ### Install Phantom

    Download from [phantom.app](https://phantom.app) for your browser or mobile device. Create a new wallet or import an existing one.
  </Step>

  <Step>
    ### Receive or Add ARIO

    ARIO will appear automatically once you receive tokens. To add manually:
    1. Open your Phantom wallet
    2. Click **"Manage Token List"** or search for tokens
    3. Search for **"ARIO"** or paste the mint address: `<ARIO_MINT_ADDRESS>`
    4. Toggle the token on
  </Step>

  <Step>
    ### Verify Your Balance

    Your ARIO balance appears alongside your other Solana tokens. You can send, receive, and swap ARIO directly within Phantom.
  </Step>
</Steps>

### Solflare

[Solflare](https://solflare.com) is a full-featured Solana wallet with built-in staking support.

<Steps>
  <Step>
    ### Install Solflare

    Download from [solflare.com](https://solflare.com) as a browser extension, web app, or mobile app.
  </Step>

  <Step>
    ### Add ARIO Token

    1. Navigate to your token list
    2. Search for **"ARIO"** or use the mint address: `<ARIO_MINT_ADDRESS>`
    3. Add the token to your portfolio
  </Step>
</Steps>

### Backpack

[Backpack](https://backpack.app) is a multi-chain wallet with strong Solana support.

<Steps>
  <Step>
    ### Install Backpack

    Download from [backpack.app](https://backpack.app) for your browser or mobile device.
  </Step>

  <Step>
    ### Add ARIO Token

    ARIO will auto-detect when tokens are received. To add manually, search for the token or paste the mint address.
  </Step>
</Steps>

## ARIO Token Details

| Property | Value |
|----------|-------|
| **Token Standard** | SPL Token |
| **Network** | Solana |
| **Mint Address** | `<ARIO_MINT_ADDRESS>` |
| **Decimals** | 6 |
| **Symbol** | ARIO |

## Managing ARIO Tokens

### Sending ARIO
1. Select ARIO from your token list
2. Click **"Send"**
3. Enter the recipient's Solana wallet address and amount
4. Confirm the transaction (requires a small amount of SOL for fees)

### Receiving ARIO
1. Share your Solana wallet address or QR code with the sender
2. Incoming tokens will appear automatically

### Swapping ARIO
You can swap ARIO on Solana DEXs like [Jupiter](https://jup.ag) or [Raydium](https://raydium.io) directly from your wallet.
