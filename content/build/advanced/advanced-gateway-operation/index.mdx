## Prerequisites

Before diving into advanced operations, ensure you have:

- A running AR.IO Gateway (see [Run a Gateway for AR.IO Network](/build/run-a-gateway/run-a-gateway-for-ario-network))
- Basic understanding of Docker and environment variables
- Administrative access to your gateway server
- Familiarity with command-line operations
