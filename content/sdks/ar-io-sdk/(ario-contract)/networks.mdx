---
title: "Networks"
description: "TypeScript/JavaScript SDK for interacting with the ar.io ecosystem"
---

The SDK provides program addresses for the mainnet and testnet environments:

- `ARIO_MAINNET_PROGRAM_ID` — Solana mainnet program address (production)
- `ARIO_TESTNET_PROGRAM_ID` — Solana devnet program address (testing and development)

As of `v3.23` the SDK defaults all API interactions to **Solana mainnet**.

#### Mainnet

```typescript
import { ARIO } from "@ar.io/sdk";

const ario = ARIO.mainnet(); // or ARIO.init()
```

#### Testnet

```typescript
import { ARIO } from "@ar.io/sdk";

const testnet = ARIO.testnet();
```
