#### spawn()

Mints a new ANT (Arweave Name Token) as a Metaplex Core NFT on Solana. This static function creates a new ANT and returns its mint address.

_Note: Requires `signer` to be provided to sign the mint transaction._

```typescript
import { ANT } from '@ar.io/sdk';

const mintAddress = await ANT.spawn({
  signer: solanaKeypairSigner,
  state: {
    name: 'My ANT',
    ticker: 'MYANT',
    description: 'My custom ANT token',
  },
});
```

**CLI Usage:**

```bash
ar.io spawn-ant --name "My ANT" --ticker "MYANT"
```

**Parameters:**

- `signer: SolanaSigner` - The signer used to authenticate the mint transaction
- `state?: SpawnANTState` - Optional initial state for the ANT including name, ticker, description, etc.
- `logger?: Logger` - Optional logger instance

**Returns:** `Promise<string>` - The mint address of the newly created ANT (Metaplex Core NFT)
