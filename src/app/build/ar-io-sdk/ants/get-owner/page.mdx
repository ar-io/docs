# getOwner

`getOwner` is a method on the `ANT` class that retrieves the owner of the configured ANT process.

`getOwner` does not require authentication.

## Parameters

`getOwner` does not accept parameters.

## Examples

<CodeGroup title="getOwner">

    ```typescript {{ title: 'NodeJS' }}
    const { ANT } = require('@ar.io/sdk');

    async function main() {
      const ant = ANT.init({ processId: "bh9l1cy0aksiL_x9M359faGzM_yjralacHIUo8_nQXM" });
      const owner = await ant.getOwner();

      console.log(owner);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ANT } from '@ar.io/sdk/web';

    async function main() {

    }

    main();
    ```
