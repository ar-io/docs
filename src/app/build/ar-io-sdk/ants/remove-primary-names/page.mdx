# removePrimaryNames

`removePrimaryNames` is a method on the `ANT` class that removes primary names from the ANT process.

`removePrimaryNames` requires authentication.

## Parameters

## Examples

<CodeGroup title="removePrimaryNames">

       ```typescript {{title: "NodeJS"}}
        const fs = require("fs");
        const { ANT, ArweaveSigner } = require("@ar.io/sdk");

        async function main() {
        const jwk = JSON.parse(fs.readFileSync("KeyFile.json"));
        const ant = ANT.init({
            signer: new ArweaveSigner(jwk),
            processId: "bh9l1cy0aksiL_x9M359faGzM_yjralacHIUo8_nQXM"
        });

        const { id: txId } = await ant.removePrimaryNames({
                names: ['arns', 'test_arns'], // any primary names associated with a base name controlled by this ANT will be removed
                ioProcessId: ARIO_TESTNET_PROCESS_ID,
            });
        }

        main();

    ```
