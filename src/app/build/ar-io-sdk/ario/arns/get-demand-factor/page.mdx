# getDemandFactor

`getDemandFactor` is a method on the `ARIO` class that retrieves the current demand factor of the network. The demand factor is a multiplier applied to the cost of ArNS interactions based on the current network demand.
`getDemandFactor` does not require authentication.

## Parameters

`getDemandFactor` does not accept parameters.

## Examples

<CodeGroup title="getDemandFactor">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const demandFactor = await ario.getDemandFactor();
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
     const demandFactor = await ario.getDemandFactor();
    }

    main();
    ```
