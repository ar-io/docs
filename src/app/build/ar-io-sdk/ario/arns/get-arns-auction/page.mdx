# getArNSAuction

`getArNSAuction` is a method on the `ARIO` class that retrieves the auction data for the specified auction name.

`getArNSAuction` does not require authentication.

## Examples

<CodeGroup title="getArNSAuction">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const auction = await ario.getArNSAuction({ name: 'permalink' });

      console.log(auction);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const auction = await ario.getArNSAuction({ name: 'permalink' });

      console.log(auction);
    }

    main();
    ```

</CodeGroup>
