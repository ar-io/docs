# getArNSRecord

`getArNSRecord` is a method on the `ARIO` class that retrieves the record info of the specified ArNS name.

`getArNSRecord` does not require authentication.

## Examples

<CodeGroup title="getArNSRecord">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const record = await ario.getArNSRecord({ name: 'ardrive' });

      console.log(record);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const record = await ario.getArNSRecord({ name: 'ardrive' });

      console.log(record);
    }

    main();
    ```

</CodeGroup>
