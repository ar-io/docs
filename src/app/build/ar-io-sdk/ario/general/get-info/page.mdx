# getInfo

`getInfo` is a method on the `ARIO` class that retrieves information about the ARIO process.

`getInfo` does not require authentication.

## Parameters

`getInfo` does not accept Parameters

## Example

<CodeGroup title="getInfo">

    ```typescript {{title: 'NodeJS'}}
    const { ARIO } = require('@ar.io/sdk');

    async function main(){
        const ario = ARIO.init();
        const info = await ario.getInfo()

        console.log(info)
    }

    main()
    ```
    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const info = await ario.getInfo()

      console.log(info);
    }

    main();
    ```

</CodeGroup>
