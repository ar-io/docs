# getCurrentEpoch

`getCurrentEpoch` is a method on the `ARIO` class that returns the current epoch data.

`getCurrentEpoch` does not require authentication.

## Parameters

`getCurrentEpoch` does not accept parameters.

## Examples

<CodeGroup title="getCurrentEpoch">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const epoch = await ario.getCurrentEpoch();

      console.log(epoch);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const epoch = await ario.getCurrentEpoch();

      console.log(epoch);
    }

    main();
    ```

</CodeGroup>
