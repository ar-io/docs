# getEpoch

`getEpoch` is a method on the `ARIO` class that returns the epoch data for the specified block height. If no epoch index is provided, the current epoch is used.

`getEpoch` does not require authentication.

## Parameters

## Examples

<CodeGroup title="getGateway">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const epoch = await ario.getEpoch({ epochIndex: 0 });

      console.log(epoch);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const epoch = await ario.getEpoch({ epochIndex: 0 });

      console.log(epoch);
    }

    main();
    ```

</CodeGroup>
