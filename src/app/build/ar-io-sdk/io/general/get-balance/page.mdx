# getBalance

`getBalance` is a method on the `IO` class that retrieves the balance of the specified wallet address.

`getBalance` does not require authentication.

## Examples

<CodeGroup title="getBalance">

    ```typescript {{title: 'NodeJS'}}
    const {IO, mIOToken} = require('@ar.io/sdk');

    async function main(){
        const io = IO.init();
        const balance = await io.getBalance({
        address: 'QGWqtJdLLgm2ehFWiiPzMaoFLD50CnGuzZIPEdoDRGQ',
        })
        .then((balance: number) => new mIOToken(balance).toIO()); // convert it to IO for readability

        console.log(balance)
    }

    main()
    ```
