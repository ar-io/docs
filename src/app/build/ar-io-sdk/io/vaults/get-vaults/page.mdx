# getGateways

`getGateways` is a method on the `IO` class that retrieves registered gateways from the ar.io Network. Results are paginated, and can be sorted by any attribute of the gateway object.

`getGateways` does not require authentication.

## Example

<CodeGroup title="getVaults">

    ```javascript {{ title: 'NodeJS' }}
    const { IO } = require('@ar.io/sdk');

    async function main() {
      const io = IO.init();
      const vaults = await io.getVaults({
        cursor: '0',
        limit: 100,
        sortBy: 'balance',
        sortOrder: 'desc',
      });

      console.log(vaults);
    }

    main();
    ```

