# getVault

`getVault` is a method on the `IO` class that retrieves the locked-balance user vault of the IO process by the specified wallet address and vault ID.

`getVault` does not require authentication.

## Examples

<CodeGroup title="getVault">

    ```typescript {{ title: 'NodeJS' }}
    const { IO } = require('@ar.io/sdk');

    async function main() {
      const io = IO.init();
      const vault = await io.getVault({
        address: 'QGWqtJdLLgm2ehFWiiPzMaoFLD50CnGuzZIPEdoDRGQ',
        vaultId: 'vaultIdOne',
      });

      console.log(vault);
    }

    main();
    ```

